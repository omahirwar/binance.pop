How the Contribution Works

POP uses a tier-based contribution model. Each tier has a defined minimum contribution amount and a fixed daily projection shown for transparency.

  • Contribution is made once per tier
  • Daily projections are calculated automatically
  • Higher tiers offer higher daily projections
  • All figures shown are for understanding participation structure

Example

If you participate in the Gold tier with $170, the system shows a projected daily return of $2.9, which equals $87 over 30 days.
